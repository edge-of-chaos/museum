# Logic & Emergence - System Archive

このリポジトリは、数理モデルと地理空間データを用いて、自律的な動態を**「ありのままに観測する」**ことを目的としたアーカイブです。
環境変数の変化によって系全体がどのように移り変わるのか、そして個別のユニットが織りなすパターンの中にどのような秩序が現れるのかを試行・記録しています。

---

## 📍 構成マップ (Directory Structure)

当サイトは以下の階層で構成されており、それぞれが論理的な繋がりを持っています。

* **[/ (Root)](https://edge-of-chaos.github.io/)**: システム・ゲートウェイ。全観測領域への入り口です。敷地と呼んでいます。
* **[/museum/](https://edge-of-chaos.github.io/museum/)**: プロジェクト・コア。各数理モデルを集約した「純粋美術館」本館です。玄関と呼んでいます。
    * **[/terrain-water/](./museum/terrain-water/)**: 標高データに基づく流体シミュレーション。物理的な勾配によって導かれる「流れ」の可視化。
    * **[/prisoners-dilemma/](./museum/prisoners-dilemma/)**: ゲーム理論的競争の動態観測。生存戦略の拡散と均衡プロセスの記録。
    * **[/election/](./museum/election/)**: 意志の階層構造シミュレーション。局所的な選択が全体構造へ波及する連鎖の記述。
    * **[/solar-basho/](./museum/solar-basho/)**: 天文幾何学による太陽高度観測。宇宙的境界と地表の受照強度のリアルタイム算出。

## ⚙️ 数理・技術的アプローチ (System Logic)

* **客観的な基底**: 恣意性を排除するため、シミュレーションの土台には国土地理院の標高データ（DEM）や天文幾何学の公認数式を採用しています。
* **動的均衡の描写**: 「重力と抵抗」「協力と裏切り」「意志と構造」といった対立する要素が交差する地点に現れる変化を、リアルタイムに記述することを目指しています。
* **対話的な観測**: パラメーターの介入によって系の安定が揺らぎ、新たな秩序（Emergence）が生まれるプロセスを、観測者が直接体験できるよう設計しています。
