# Logic & Emergence - System Archive

このリポジトリは、数理モデルと地理空間データを用いて、自律的な動態を「ありのままに観測する」ことを目的としたアーカイブです。
環境変数の変化によって系全体がどのように移り変わるのか、そして個別のユニットが織りなすパターンの中にどのような秩序が現れるのかを試行・記録しています。

## 📍 構成マップ (Directory Structure)

当サイトは以下の階層で構成されており、それぞれが論理的な繋がりを持っています。

* **[/ (Root)](https://edge-of-chaos.github.io/)**: システム・ゲートウェイ。全観測室への入り口です。
* **[/museum/](https://edge-of-chaos.github.io/museum/)**: プロジェクト・コア。各数理モデルを集約した本館です。
    * **[/museum/terrain-water/](https://edge-of-chaos.github.io/museum/terrain-water/)**: 標高データに基づく流体シミュレーション。物理的な勾配によって導かれる「流れ」を可視化しています。
    * **[/museum/prisoners/](https://edge-of-chaos.github.io/museum/prisoners/)**: 2次元格子上のゲーム理論的競争。生存戦略がどのように広がり、均衡していくかを観測します。

## ⚙️ 数理・技術的アプローチ (System Logic)

1. **客観的な基底**: 恣意性を排除するため、シミュレーションの土台には国土地理院が提供する実在の標高データ（DEM）を採用しています。
2. **動的均衡の描写**: 「重力と抵抗」や「協力と裏切り」といった対立する要素が交差する地点に現れる変化を、リアルタイムに記述することを目指しています。
3. **対話的な観測**: 降雨率や突然変異率といったパラメーターを観測者が操作することで、系全体の安定性がどのように揺らぐのかを体験的に理解できるよう設計しています。

## 📝 外部データ利用と謝辞 (Credits & Etiquette)

本プロジェクトの構築にあたり、以下の優れたリソースを活用させていただきました。貴重なデータとツールの提供に、深く感謝の意を表します。

* **地理院タイル (国土地理院)**: 地形シミュレーションの核となる高精度な標高データの出典として利用しています。
* **OpenStreetMap (Nominatim)**: 地名から座標を取得する検索機能においてAPIを利用しています。
* **Tailwind CSS / Chart.js**: インターフェースのデザインおよび統計データの可視化に活用しています。
