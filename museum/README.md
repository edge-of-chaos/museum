# Museum of Logic & Emergence: Core Archive

ここは、数理的なアルゴリズムが描く「動的な秩序」を集約した空間です。
実在のデータや物理的な法則に基づいた複数の展示室（シミュレーター）を通じて、複雑な現象の背後にある簡潔な論理を観測することを目的としています。

## 🏛 展示室案内 (Exhibits)

当初、以下の2つの観測装置が稼働していますが、随時追加予定としております。

### 1. [Terrain-Water Flow Simulation](./terrain-water/)
* **概要**: 国土地理院の標高データを基底とした、地表における流体の軌跡の観測。
* **論理**: 重力と勾配という物理的な制約によって、水がどのように「道」を選択し、地形がそれを規定するかを記述します。
* **観測の視点**: 作為のない物理法則が、いかにして美しい網目状の構造を創発させるかに着目してください。

### 2. [Social Interaction: Action-Intuition](./prisoners/)
* **概要**: エージェント・ベース・モデリングによる戦略的競争の動態観測。
* **論理**: 個別のユニットが持つシンプルなルール（戦略）が、相互作用を通じてどのように広域的な分布や均衡を形成するかを可視化します。
* **観測の視点**: 利己と互恵、突然変異と継承が、系全体の多様性にどのような揺らぎをもたらすかを追跡します。

## 🔍 観測にあたって (General Principle)

当館の展示は、単なる情報の提示ではなく、パラメーターへの介入を通じた「対話的な観測」を重視しています。
* **変数の操作**: 降雨率や突然変異率を変更することで、系の安定性が失われる瞬間や、新たな秩序が生まれるプロセスを観察できます。
* **データの誠実性**: 全ての描画は、加工されていない数理的出力に基づいています。
