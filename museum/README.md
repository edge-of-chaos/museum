# Museum of Logic & Emergence: Core Archive

ここは、数理的なアルゴリズムが描く**「動的な秩序」**を集約した空間です。
実在のデータや物理的な法則に基づいた複数の展示室（シミュレーター）を通じて、複雑な現象の背後にある簡潔な論理を観測することを目的としています。

---

## 🏛 展示室案内 (Exhibits)

現在、当館では以下の4つの観測装置が稼働しており、世界の多層的な側面を記述しています。

### 1. Terrain-Water Flow Simulation
* **概要**: 国土地理院の標高データを基底とした、地表における流体の軌跡の観測。
* **論理**: 重力と勾配という物理的な制約によって、水がどのように「道」を選択し、地形がそれを規定するかを記述します。
* **観測の視点**: 作為のない物理法則の結果と、実在の川を見比べて楽しみましょう。

### 2. Social Interaction: Action-Intuition (Dialogue of Mirrors)
* **概要**: エージェント・ベース・モデリングによる戦略的競争の動態観測。
* **論理**: 個別のユニットが持つシンプルなルール（戦略）が、相互作用を通じてどのように広域的な分布や均衡を形成するかを可視化します。
* **観測の視点**: 利己と互恵、その継承が、系全体の多様性にどのような揺らぎをもたらすかを追跡します。

### 3. Hierarchy of Wills: Structural Dynamics
* **概要**: 多層的な組織構造における「意志」の伝播と再編のシミュレーション。
* **論理**: 下位層の意志決定が上位層の重みを規定。構造の中での「一票の重み」を実感できます。
* **観測の視点**: 個々の微小な意志が、構造というフィルターを通った際にどのような「全体の総意」へと変質するのかを観測します。

### 4. Temporal Space: The Boundary Observer
* **概要**: 天文幾何学に基づく太陽高度と、地表が受ける相対光量強度のリアルタイム観測。
* **論理**: ランベルトの余弦則を適用し、地球上の任意地点における光の強さを算出。日の出、日の入りなどの境界を通して、宇宙の律動を刻みます。
* **観測の視点**: 幾何学的な「高度」が、いかにして私たちの生存環境としての「明るさ」や「境界」を規定しているかを静かに見つめます。

## 🔍 観測にあたって (General Principle)

当館の展示は、単なる情報の提示ではなく、パラメーターへの介入を通じた**「対話的な観測」**を重視しています。

* **変数の操作**: あなたの操作によって、系の安定が揺らいだり、新たな秩序が生まれるプロセスを直接観察できます。
* **データの誠実性**: 全ての描画は、加工されていない数理的出力、または公的な地理・天文データに基づいています。
